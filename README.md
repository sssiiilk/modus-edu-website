# Городской колледж «МОДУС»

Лендинг и информационный сайт колледжа: специальности, приёмная кампания, инфраструктура и онлайн-заявка.

## Стек

- **React 19** + **Vite 8**
- Inline-стили и тема через React Context (`src/lib/theme.jsx`)
- Адаптив через хук `useBreakpoint` (mobile / tablet / desktop)
- Без React Router — страницы переключаются состоянием в `App.jsx`

## Страницы

| Страница | Описание |
|----------|----------|
| Главная | Герой, статистика, направления, инфраструктура, новости, форма обратной связи |
| Специальности | Фильтрация, поиск, сортировка и карточки программ |
| Приёмная кампания | Обзор сроков, документы, форма заявки, FAQ |

## Быстрый старт

```bash
npm install
npm run dev
```

Приложение откроется на `http://localhost:5173`.

### Скрипты

| Команда | Назначение |
|---------|------------|
| `npm run dev` | Локальная разработка с HMR |
| `npm run build` | Сборка в `dist/` |
| `npm run preview` | Просмотр production-сборки |
| `npm run lint` | ESLint |

## Структура

```
src/
  App.jsx                 # Тема, шапка/подвал, переключение страниц
  components/
    SiteHeader.jsx        # Навигация + мобильное меню
    SiteFooter.jsx
  pages/
    HomePage.jsx
    ProgramsPage.jsx
    AdmissionsPage.jsx
  lib/
    theme.jsx             # Цвета, шрифты, радиусы, тени
    ui.jsx                # Heading, Button, Card, Container…
    useBreakpoint.js      # Брейкпоинты
    programs.js           # Данные специальностей
    icons.jsx
  index.css
public/
  assets/                 # Логотип и изображения кампуса
```



## Примечание

Формы заявки и обратной связи работают на клиенте, бэкенд для отправки данных не подключён.
